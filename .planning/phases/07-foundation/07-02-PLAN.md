<objective>
Add project-scoped memory enforcement: every memory write requires a project_id, global writes need explicit cross_project flag, keys use project prefix format, and a pollution scan detects cross-project contamination at startup.

Purpose: Prevents cross-project memory contamination (the #1 risk identified in research). Without project scoping, memories from different projects bleed into retrieval, causing incorrect context injection.
Output: Updated `lib/aegis-memory.sh` with scoped functions, updated taxonomy docs, comprehensive test file.
</objective>
